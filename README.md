## 🏗️ Featured Projects

### 🍎 Foodbank Check-In System
**Capstone Project | Production System | Full-Stack**

React • TypeScript • Node.js • Express • Supabase • PostgreSQL • Vercel • Railway

A complete full-stack check-in and appointment system built solo as my Bachelor of Computing Science capstone project. **25,135 lines of production-ready code** serving real food bank operations.

**Key Features:**
- QR code-based client check-in workflow
- Real-time admin dashboard with analytics
- 7-language internationalization (i18n)
- CSV-first architecture (Link2Feed compatible)
- Auto-scheduled appointments (21+ days ahead)
- Privacy-compliant (24-hour auto-purge)
- WCAG 2.1 AA accessible

**Impact:**
- Reduced check-in time from 15+ minutes to under 5 minutes
- Production-deployed and actively used
- Multi-location support with data isolation
- Graduated with **Distinction** as the only mother in the program

**Live Demo:**

- [Admin Dashboard](https://foodbank-checkin.vercel.app/login)
